# Ideas para el juego de las hormigas

Se me ocurre que, la idea que había dicho antes de seleccionar una hormiga y asignarles acciones no es viable porque serán demasiadas hormigas.

Pero si es viable tener múltiples eventos aleatorios que afecten el dinamismo del juego.

Cómo en age of empire se construyen casas para tener más población, al apretar el nido se podrían comprar más túneles con materiales, para tener más hormigas

--------------------------

Inicio:

- Inundacion del hormigero 
- Invacion de otros hormigeros, guerra campal
- Bichos voladores
